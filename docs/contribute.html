Contribuir

Você pode contribuir com quaisquer dados de conteúdo de ferro do solo, seja qual for o método analítico usado, a forma e profundidade de amostragem, ou a localização geográfica das amostras analisadas, desde que esses dados sejam fruto de suas atividades e/ou estejam sob sua responsabilidade. Mas antes de preparar seu conjunto de dados para contribuição, recomendamos que você leia atentamente as diretrizes abaixo e, caso restarem dúvidas, entre em contato conosco. Assim poderemos discutir detalhes importantes e garantir que o seu (e o nosso) tempo seja gasto de maneira racional.

Forma de Submissão

Os dados podem ser submetidos de várias maneiras:

  • Caso os dados estejam no formato analógico (relatórios, dissertações e teses), pedimos que você faça a sua digitalização usando o nosso modelo de planilha eletrônica que contém nomes padronizados para as variáveis.
  • Se os dados já estiverem no formato digital e o número de registros for pequeno (< 1000 registros), pedimos que os mesmos sejam organizados usando também o nosso modelo de planilha eletrônica.
  • Mas se você tiver um volume muito grande de dados (> 1000 registros) já no formato digital, envie-nos uma mensagem para estudarmos a forma mais apropriada para o sua submissão.

Nossos modelos de planilha eletrônica para organização dos dados foram construídos usando os mesmos padrões internacionais usados pelo World Soil Information Service (WoSIS). Versões para usuários de LibreOffice (formato ODS) e Microsoft Office (formato XLS) podem ser descarregadas de:

  • LibreOffice: https://github.com/samuel-rosa/Fe-BR/raw/master/templates/template.ods
  • Microsoft Office: https://github.com/samuel-rosa/Fe-BR/raw/master/templates/template.xls
